os seletores são

seletor de elemento

O seletor de elemento é o tipo mais basico, ele seleciona os elementos html como <h3>
Neste caso está sendo usada para adicionar o underline nas tags

seletor de classe

O seletor de classe serve para selecionar as classes que são definidas nas tags, como uma tag <section> com classe .paragrafos
Que neste caso está sendo usada para alinhar os paragrafos a esquerda

seletor de id

O seletor de id, diferente do de classe, serve para selecionar algum elemento único, chamado por #
neste caso está sendo usado para deixar o texto verde

seletor de atributo

O seletor de atributo chama elementos que possuem um atributo especifico, por exemplo um href que possui um link especifico
neste caso está alterando a cor do link abaixo para preto

infnet.edu

seletor de descendente

O seletor de descendente chama elementos que são descendentes de um outro elemento "pai"
Neste caso está sendo usado para alterar a fonte od texto para cursiva

seletor de pseudo-classe

Esse seletor é ativado quando o elemento atinge algum estado em especifico, como por exemplo quando o mouse passa por cima do elemento pose se ativar a pseudo classe hover
neste caso está fazendo com que o titulo dessa explicação seja destacada em azul quando selecionada
